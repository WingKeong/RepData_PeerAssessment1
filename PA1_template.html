    Missing values will be replaced by the average number of steps (rounded to integers) for each interval (avint), calculated in the previous question

  2. Create a new dataset that is equal to the original dataset but with the missing data filled in.

df1$wd<-factor(ifelse(weekdays(as.Date(df1$date),T) %in% c("Sat","Sun"),"Weekend","Weekday"))

AWk<-aggregate(steps~wd+interval, data=df1,FUN=mean)
